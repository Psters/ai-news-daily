# AI News Daily

每日 AI 前沿动态追踪 - 2026.03.20

## 项目简介

这是一个现代化的 AI 新闻展示页面，使用前端设计最佳实践构建，展示最新的 AI 行业动态。

## 特性

- 🎨 现代化设计：编辑/杂志风格，深色主题
- 📱 响应式布局：移动优先，适配各种屏幕尺寸
- ⚡ 性能优化：CSS 动画，平滑滚动
- 🔍 数据来源：Tavily Search API
- 🎯 设计理念：避免通用 AI 美学，注重独特性和可读性

## 技术栈

- HTML5 语义化标签
- CSS Grid + Flexbox 布局
- CSS 变量主题系统
- 原生 JavaScript（无框架依赖）
- 字体：Noto Serif SC + JetBrains Mono

## 部署

本项目通过 GitHub Actions 自动部署到 GitHub Pages。

Live site: https://psters.github.io/ai-news-daily/

## 设计理念

基于 frontend-design-3 Skill 的设计原则：

- 避免通用字体（Inter、Roboto、Arial）
- 避免陈词滥调的配色（紫色渐变 + 白色背景）
- 选择大胆的美学方向：编辑/杂志风格
- 强调对比和可读性
- 深色主题 + 亮色文字
- 现代 CSS（Grid、Flexbox、CSS 变量）
